문의하기

서비스 개선 아이디어, 오류 제보, 기능 제안 등을 자유롭게 남겨 주세요.

문의 안내

내 연봉 아파트 계산기는 1인 개발자가 운영하는 프로젝트로, 모든 문의에 실시간으로 답변드리기는 어렵지만 가능한 범위 내에서 성실하게 의견을 검토하고 반영하려고 노력하고 있습니다. 계산 결과가 실제 은행 상담 내용과 많이 다르거나, 규제 적용 방식에 오류가 의심되는 경우, 혹은 추가로 있었으면 하는 기능이 있다면 편하게 알려 주세요.

연락처

· 이메일: example@example.com
· GitHub 이슈: 배포 후 저장소 URL 기입

위 연락처는 예시이며, 실제 운영에 사용하는 이메일 또는 GitHub 저장소 주소로 교체해 주세요.

문의 양식 (예시)

실제로 서버에 저장되거나 메일이 전송되지는 않고, 사용자가 어떤 내용을 남기면 좋을지 참고용으로 작성된 양식입니다.

1) 문의 유형: 오류 신고 / 기능 제안 / 기타
2) 사용 환경: OS, 브라우저, 모바일/PC 여부 등
3) 자세한 내용: 재현 경로, 기대한 결과, 실제 결과 등
4) 연락 가능 메일 주소(선택)
