월급쟁이가 진짜 쓰는 AI 도구 BEST 5

2026-06-16 | 작성자: 승박

부업·재테크에 바로 써먹는 활용법 — "AI 도구가 좋다는 건 알겠는데, 도대체 뭐부터 써야 하나요?" 저도 똑같은 고민을 했습니다. 오늘은 제가 실제로 매일 쓰면서 효과 봤던 AI 도구 5개를 솔직하게 공유합니다. 단순한 소개가 아니라, 어떤 상황에 뭘 쓰는지까지 정리했어요.

평일엔 생산관리 일 하고, 퇴근하면 부동산 공부하고, 주말엔 유튜브 콘텐츠 만들고. 시간이 절대적으로 부족한 월급쟁이에게 AI 도구는 '있으면 좋은 것'이 아니라 '반드시 써야 하는 것'이더라고요.

1. ChatGPT — 만능 입문기, 일단 이거부터

가장 유명한 만큼 가장 무난합니다. 처음 AI를 접하는 분이라면 무조건 ChatGPT부터 시작하세요.

제가 쓰는 방식:

  • 이메일 초안 작성 (특히 영어 이메일)
  • 회의록 요약 및 정리
  • 아이디어 브레인스토밍
  • 모르는 용어 쉽게 설명받기

유료 구독 가치 있나? 무료로도 충분히 강력합니다. 하지만 매일 30분 이상 쓴다면 월 2만 원대 유료 플랜이 시간 대비 가치가 확실합니다.

2. Claude — 글쓰기와 긴 문서 분석의 끝판왕

저는 블로그 글 초안, 유튜브 스크립트, 그리고 부동산 공부할 때 법령이나 PDF 문서 요약에 Claude를 씁니다. ChatGPT보다 글이 훨씬 자연스럽고, 한 번에 처리할 수 있는 문서 양이 압도적으로 많습니다.

실제 사용 예시: 부동산 실거래가 공공데이터 API 문서(영문 50페이지)를 Claude에 통째로 넣고 "한국어로 요점만 정리해줘"라고 하면 5초 만에 완벽한 요약본을 줍니다.

장점: 코딩과 글쓰기에서 ChatGPT보다 우위. 긴 문서 한 번에 처리 가능.

단점: 이미지 생성 안 됨. 웹 검색 약함.

3. Cursor — 코딩 1도 몰라도 사이트 만드는 AI 코드 에디터

이건 제가 가장 충격받은 도구입니다. 저는 코딩을 전혀 모르는 월급쟁이인데, Cursor 덕분에 부동산 실거래가 사이트를 만들고 있습니다.

Cursor가 뭔가요? AI가 내장된 코드 에디터입니다. 한국어로 "강남구 아파트 실거래가 보여주는 코드 만들어줘"라고 입력하면 코드를 자동으로 작성해줍니다.

이런 분에게 추천:

  • 나만의 사이트나 도구를 만들고 싶은 직장인
  • 외주 비용 아끼고 싶은 소상공인
  • 코딩 입문하고 싶지만 막막한 분

저는 Cursor + ChatGPT/Claude 조합으로 사이트 개발 비용 수백만 원을 아꼈습니다. 진짜로요. (관련 글: AI로 만든 이 사이트 이야기)

4. Perplexity — 검색 시간을 90% 줄여주는 AI 검색엔진

"네이버 광고 너무 많아서 진짜 정보 찾기 힘들어요."

저도 그랬습니다. 그래서 요즘은 Perplexity를 씁니다. 질문을 하면 출처와 함께 정리된 답변을 줍니다. 광고도 없고, 근거도 명확합니다.

저는 이럴 때 씁니다:

  • 부동산 시세 트렌드 조사
  • 경제 뉴스 빠르게 파악
  • 유튜브 콘텐츠 자료 조사

5. Gemini — 구글 생태계 쓰는 사람의 비밀 무기

Gmail, Google Docs, Google Sheets를 쓰는 분이라면 Gemini가 유리합니다. 구글 워크스페이스에 깊게 통합되어 있어서 "이번 달 가계부 정리해줘" 같은 명령을 Sheets에서 바로 실행할 수 있습니다.

월급쟁이의 솔직한 추천 조합

5개 다 쓰라는 건 비현실적입니다. 제가 추천하는 단계별 조합은:

  • 입문 (월 0원): ChatGPT 무료 + Perplexity 무료
  • 중급 (월 2만 원): Claude Pro 또는 ChatGPT Plus 하나
  • 고수 (월 4만 원): Claude Pro + Cursor Pro
처음부터 5개 다 깔 필요 없습니다. 무료 2개로 시작해서, 진짜 막히는 순간에 하나씩 추가하세요.

마치며

AI 도구는 '대체'가 아니라 '확장'입니다. 시간이 부족한 월급쟁이일수록 AI를 어떻게 활용하느냐가 향후 5년의 격차를 만들 거라고 확신합니다.

저도 매일 새로운 활용법을 발견하고 있습니다. 이 블로그에서 계속 공유할 테니 자주 들러주세요.
